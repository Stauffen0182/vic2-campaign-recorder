# How to use optimally
1. Set autosave to monthly (only the host in mp gets autosaves!)
2. Set vic2 savefolder
3. Press "create new campaign" & make sure its the right mod
4. Press "start recording"
5. Press "end recording" at the end of session
6. Press "export interactive (HTML)" to see data

# Reviewing saves without recording
Non-autosaves - like old campaigns - can also be exported.

These will work fine (except for monthly growth), but expect less smooth graphs.

1. Set vic2 savefolder
2. Press "create new campaign" & make sure its the right mod
3. Drag old saves into savegames\vic2_save_recorder\campaigns(yourcampaign)\saves
4. Press "reprocess all saves" (1-5 minutes)
5. Press "export interactive (HTML)" to see data
